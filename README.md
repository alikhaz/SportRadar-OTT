# SpOTT challange

## Folders

- Front: Frontend code in Vue.js framework
  - Techs:
    - Vue.js (Typescript, Sass)
    - Vuex
    - Bootstrap (CSS only)
- Back: Backend system in Laravel platform
  - Techs:
    - Laravel (Headless rest API)

---

## HowTos

### Frontend

From folder:

```
front/app/
```

Running on http://localhost:8080

```
yarn serve
```

Running tests

```
yarn test
```

&nbsp;

### Backend

From folder

```
back/
```

Running on http://localhost:8000

```
php artisan serve
```

Running tests

```
php artisan test
```
